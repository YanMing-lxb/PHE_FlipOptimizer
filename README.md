
# PHE_DraftOptimizer
板式换热器翻边计算工具

## 项目简介

PHE_FlipOptimizer（板式换热器翻边计算工具）是一个专门用于优化板式换热器制造过程中翻边、翻转和拔模工艺的计算工具。该工具可以帮助工程师快速准确地计算出最优的翻边参数，从而提高工艺设计效率和产品质量。

## 功能特点

- 自动计算翻边角度和拔模角度
- 计算板片长度、宽度和圆角半径
- 提供友好的命令行交互界面

## 安装与运行

### 运行环境

- Python 3.x

### 运行方法

```bash
cd src
python main.py
```
或直接运行 exe 文件

按照提示输入相关参数即可获得优化结果。

## 使用说明

运行程序后，根据提示依次输入以下参数：

1. A侧通道间距 (mm)
2. B侧通道间距 (mm)
3. 板片厚度 (mm)
4. 板片长度 (mm)
5. 板片宽度 (mm)
6. 圆角半径 (mm)

程序会自动计算并输出优化结果，包括：

- 翻边角度
- 拔模角度
- 优化后板片长度
- 优化后板片宽度
- 优化后圆角半径

## 注意事项

优化后的尺寸指的是通道间距较大的板片。
